# Security Policy

The Musette MCP server is stateless and public by design: it holds no user accounts, stores no
personal data, and processes no payments. The order link it returns carries only product ids and
quantities; real prices are resolved server-side and re-validated at checkout on musette.cc.

## Reporting a vulnerability

Found something? Please email **info@musette.cc** with a description and reproduction steps.
Do not open a public issue for security-sensitive reports. We aim to acknowledge within 3 working
days.

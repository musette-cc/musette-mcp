# Contributing

Thanks for your interest in the Musette MCP connector.

This repository is the public face of a connector that runs as an edge function on Musette's own
infrastructure. The most useful contributions are:

- **Bug reports** — a tool returns something wrong or confusing? Open an issue with the exact prompt
  and the output you got.
- **Directory listings** — spotted an MCP directory we're not in yet? Let us know (or open a PR
  adding it to the README's *Publishing & directories* section).
- **Docs & examples** — clearer setup steps, more example prompts, translations.

For nutrition-logic changes: the carbohydrate engine is validated by sports dietitian Ien Vitse and
maintained in the main Musette codebase as a single source of truth — so substantive plan changes
are made there, not here. Open an issue to discuss before sending a PR.
